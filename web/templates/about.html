Aitana is a smart image-based classification system designed to help communities manage electronic waste in a more sustainable and creative way. This project focuses on the Eastleigh neighborhood in Nairobi, where e-waste is a growing concern due to dense population and electronics trade.

Using machine learning, Aitana identifies common discarded electronics such as mobile phones, laptops, televisions, and printers. Once identified, the system suggests practical reuse or upcycling ideas, helping users reduce waste and support a circular economy.

This project was developed as part of a Software Engineering capstone at the African Leadership University. It aims to promote environmental awareness, empower local communities, and reduce health and pollution risks associated with e-waste.

Technologies used include image classification (CNNs), FastAPI/Flask for backend deployment.
